# Frontend

Aplicación Vite para la tienda y el panel administrativo.

## Requisitos

- Node.js
- pnpm

## Instalación y ejecución

1. Entrar a la carpeta `frontend`.
2. Instalar dependencias:

```bash
pnpm install
```

3. Levantar el frontend:

```bash
pnpm run dev
```

La app queda disponible en Vite, normalmente en `http://localhost:5173`.
